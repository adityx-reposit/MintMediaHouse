**Old Flow ❌:**
Hero → Stats → Marquee → Services → Work → LaunchVideos → Growth → Testimonials → Process → Quote → BookCall → Footer

**New Flow ✅:**
```

  ↓
Client Logos (Social Proof - Built Trust Immediately)
  ↓
Stats (10M+ Views, 15+ Clients, 100% Satisfaction)
  ↓
Services (Results-Driven Copy, Not Feature-Based)
  ↓
Work/Portfolio (Showcase Quality)
  ↓
Case Studies (Real Client Results with ROI Metrics)
  ↓
Trust Signals (Guarantees, Response Time, Money-Back)
  ↓

  ↓

  ↓
Process (How We Work - Builds Confidence)
  ↓

  ↓
Simplified Lead Form (Only 3 Fields - Name, Email, Message)
  ↓
CTA (Book Free Strategy Call)
  ↓
Footer
```

**Why This Works:**
- ✅ Builds trust early (client logos at #2)
- ✅ Shows results before asking for commitment (case studies before form)
- ✅ Reduces friction (3-field form vs 9-field)
- ✅ Multiple CTAs at strategic points (pricing, FAQ, form)
- ✅ FAQ before form addresses objections preemptively

```javascript

  "@type": "Organization",
  "name": "Mint Media House",
  "url": "https://mintmediahouse.in",
  "logo": "...",
  "email": "hello@mintmediahouse.in",
  "address": {
    "@type": "PostalAddress",
    "addressLocality": "Mumbai",
    "addressCountry": "IN"
  },
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": "4.9",
    "reviewCount": "47"
  }
}

// LocalBusiness Schema
{
  "@type": "LocalBusiness",
  "name": "Mint Media House",
  "address": {...},
  "geo": {
    "@type": "GeoCoordinates",
    "latitude": "19.0760",
    "longitude": "72.8777"
  }
}

// Product/Service Schema
{
  "@type": "ProductCollection",
  "hasOfferCatalog": {
    "@type": "OfferCatalog",
    "itemListElement": [
      {
        "@type": "Offer",
        "name": "UI Animations",
        "price": "50000",
        "priceCurrency": "INR"
      },
      ...
    ]
  }
}

// FAQPage Schema
{
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "...",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "..."
      }
    },
    ...
  ]
}
```

**Benefits:**
- ✅ Rich snippets in Google Search
- ✅ Local pack visibility (Maps)
- ✅ FAQ carousel in SERPs
- ✅ Better CTR from search results
